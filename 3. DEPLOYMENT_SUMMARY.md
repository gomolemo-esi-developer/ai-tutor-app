## EXPECTED ISSUES & FIXES

### Issue: `docker-compose build` fails

**Cause**: Missing files or bad configuration

**Fix**:
```bash
# Check validation
bash VALIDATE_DEPLOYMENT.sh

# Check for specific errors
docker-compose build --no-cache
```

### Issue: Services start but can't connect to each other

**Cause**: Wrong service URLs in environment

**Docker (local)**: Use internal names
```
RAG_SERVICE_URL=http://rag-service:8000
```

**Render.com (production)**: Use external URLs
```
RAG_SERVICE_URL=https://tutorverse-rag.onrender.com
```

### Issue: Frontend shows blank page

**Cause**: Wrong backend URL or CORS issue

**Fix**:
1. Check browser console for errors
2. Update `VITE_API_URL` environment variable
3. Add CORS headers to backend:
   ```typescript
   app.use(cors({ origin: '*' }));
   ```

### Issue: RAG service health check fails

**Cause**: Missing health endpoint or wrong port

**Fix**:
Add to `main.py`:
```python
@app.get("/health")
async def health():
    return {"status": "ok"}
```
